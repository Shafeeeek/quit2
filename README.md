<h2>📱 About The Project</h2>

<p>
<b>Reboot90</b> is a modern Flutter application designed to help users quit smoking using a structured <b>90-Day Reboot Program</b>.
</p>

<p>
It combines behavioral psychology, dopamine reset principles, and habit tracking into a clean, professional mobile experience.
</p>

<ul>
  <li>🚭 Quit smoking permanently</li>
  <li>📅 Follow structured 90-day phases</li>
  <li>🔥 Track streaks & habits</li>
  <li>💰 Monitor money saved</li>
  <li>🧠 Emergency craving support</li>
</ul>

<hr/>

<h2>🎯 Core Features</h2>
